# Disclaimer

## Repository Scope

This repository contains **ONLY** the landing page and marketing website for Bridgit Social. 

### What This Repository Includes ✅

- Landing page HTML, CSS, and JavaScript
- Blog system for content publishing
- Marketing materials and design assets
- Educational components for public learning benefit
- Future placeholder structure for web app integration

### What This Repository Does NOT Include ❌

- **Bridgit Social mobile application source code** (iOS/Android)
- Backend services and API implementations
- Database schemas and migrations
- Authentication and user management systems
- Business logic and proprietary algorithms
- Production infrastructure configuration
