## 📱 About Bridgit Social

Bridgit Social is a location-based networking app that helps you discover and connect with real people, in real time, wherever you are. Whether you're at a café, coworking space, or just nearby, Bridgit makes meaningful connections effortless.

**Download on the App Store:** [Bridgit Social](https://apps.apple.com/us/app/bridgit-social/id6743941115)  
**Get it on Google Play:** [Bridgit Social](https://play.google.com/store/apps/details?id=com.bridgitsocial.android_prod)

### Key Features

- 📍 **Real-time location-based discovery**
- 🎭 **Anonymous browsing until you match**
- 🤝 **"Mark to Meet" for mutual interest**
- 🗺️ **Privacy-protected map view**
- 🔒 **Privacy & safety first**
- ✍️ **Private notes after connections**
- 🚶 **Two modes: Same Place or Walking Distance**

## 🚀 Quick Start

### Prerequisites

- [Node.js](https://nodejs.org/) >= 18.0.0
- [npm](https://www.npmjs.com/) >= 9.0.0

### Installation

```bash
# Clone the repository
git clone https://github.com/amfooladgar/bridgitsocial-landingpage.git

# Navigate to project directory
cd bridgitsocial-landingpage

# Install dependencies
npm install

# Start development server
npm run dev
```

Visit `http://localhost:5173` to see the site.

## 🛠️ Development

### Available Scripts

```bash
npm run dev          # Start development server with hot reload
npm run build        # Build for production
npm run build:blog   # Generate blog posts from markdown
npm run preview      # Preview production build
npm run deploy       # Build and deploy to GitHub Pages
npm run format       # Format code with Prettier
npm run format:check # Check code formatting
```
